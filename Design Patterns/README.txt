
These are design pattern sample code

in eclipse java

including Patterns:

Strategy -- exampled by a ship that can change its weapon(gun, bomb) to different kind
Observer -- exampled by a newspaper store that distribute newspaper to different people
Decorator -- exampled by beverage order with customized condiments
Factory -- Pizza Store(has factory for pizza initiation) Serving different kinds of Pizza
Singleton -- A GlobalSetting Class implemented as Singleton to allow only one instance.
